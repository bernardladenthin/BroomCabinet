# Snippets

Small, reusable code snippets, organized by programming language. Each language
folder keeps its snippets together in a single `README.md`.

| Language | Snippets |
|----------|----------|
| [`JavaScript/`](JavaScript/) | Nashorn object → plain object converter. |
| [`PHP/`](PHP/) | Micro-benchmark scripts (GPL / CC-BY) and small server snippets (list Apache modules, mod_rewrite check, phpinfo). |
