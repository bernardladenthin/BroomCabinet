# File endings

File-name patterns worth ignoring or deleting — drop them into a `.gitignore`,
an rsync/backup exclude list, or delete them with the generic commands below.
Platform-agnostic.

## LaTeX build artifacts

Generated by a LaTeX build and safe to delete afterwards:

```gitignore
*.aux
*.backup
*.bbl
*.blg
*.definition
*.dvi
*.idx
*.ilg
*.ind
*.info
*.lof
*.log
*.lot
*.ps
*.synctex
*.toc
*~
```

## OS / application junk files

Created by operating systems and apps, never worth keeping:

```gitignore
.DS_Store
._.DS_Store
Thumbs.db
desktop.ini
*.AAE
AlbumArt*
.picasa.ini
.picasaoriginals
~$*.docx
~WRL*.tmp
.~lock.*odt#
```

## Deleting them

Generic commands — plug in whichever patterns from the lists above you want
(`*.aux *.log …` for LaTeX, `.DS_Store Thumbs.db …` for junk).

**POSIX shell** — current directory, then a whole tree:

```sh
rm -f *.aux *.log .DS_Store Thumbs.db
find . \( -name '*.aux' -o -name '.DS_Store' -o -name 'Thumbs.db' \) -delete
```

**PowerShell** — current directory, then a whole tree:

```powershell
Remove-Item -Force *.aux, *.log, .DS_Store, Thumbs.db
Get-ChildItem -Recurse -Force -Include *.aux, *.log, .DS_Store, Thumbs.db -File | Remove-Item -Force
```
